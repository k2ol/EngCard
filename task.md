## Technical Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Storage**: LocalStorage for user progress
- **Audio**: Web Audio API for pronunciation
- **Responsive**: CSS Grid and Flexbox
- **Accessibility**: ARIA labels and semantic HTML
